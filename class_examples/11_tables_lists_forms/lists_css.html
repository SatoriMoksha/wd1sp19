HTML Lists and CSS

list-style-type

HTML

<section>
	<h2>Grocery List</h2>
	<ul>
		<li>Coffee</li>
		<li>Milk</li>
		<li>Eggs</li>
		<li>Bread</li>
	</ul>
</section>

CSS

ul {
	list-style-type: circle;
}

Output:

Grocery List

  • Coffee
  • Milk
  • Eggs
  • Bread

list-style-position

HTML

<section>
	<h2>Inside</h2>
	<ul id="inside">
		<li>This is a long sentence that will wrap</li>
	</ul>
	<h2>Outside</h2>
	<ul id="outside">
		<li>This is a long sentence that will wrap</li>
	</ul>
</section>

CSS

#inside {
  list-style-position: inside;
  width: 100px;
}
#outside {
  list-style-position: outside;
  width: 100px;
}

Output:

Inside

  • This is a long sentence that will wrap

Outside

  • This is a long sentence that will wrap
