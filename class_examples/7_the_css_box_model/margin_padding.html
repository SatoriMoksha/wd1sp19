The CSS Box Model - Padding

Collapsed Margins

HTML

<section>
	<p>This paragraph has a bottom margin of 3em.</p>
	<p>This paragraph has a top margin of 3em. The space above is only 3em, not 6em.</p>
</section>

CSS

section p:first-child {
	margin-bottom: 3em;
}
section p:last-child {
	margin-top: 3em;
}

Output:

This paragraph has a bottom margin of 3em.

This paragraph has a top margin of 3em. The space above is 3em, not 6em.

Padding Does Not Collapse

HTML

<section>
  <p>This paragraph has a bottom padding of 3em.</p>
  <p>This paragraph has a top padding of 3em. The space above is 6em, plus margins.</p>
</section>

CSS

section p:first-child {
  padding-bottom: 3em;
	background: #EBEBEB;
}
section p:last-child {
  padding-top: 3em;
	background: #BEBEBE;
}

Output:

This paragraph has a bottom padding of 3em.

This paragraph has a top padding of 3em. The space above is 6em, plus margins.

Center a Block Element

HTML

<section>
  <p>This is a paragraph.</p>
  <p>This is another paragraph.</p>
</section>

CSS

section {
	width: 400px;
	margin: 0 auto;
	padding: 1em;
	border: 1px solid black;
}

Output:

This is a paragraph.

This is another paragraph.
