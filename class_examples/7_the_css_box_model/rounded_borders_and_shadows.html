Rounded Borders and Shadows

Example

HTML

<section>
	<p>This section has a black solid border with rounded corners.</p>
	<article>
		<p>This article has a left double border with large rounded corneres.</p>
	</article>
</section>

CSS

section {
	width: 50%;
	padding: 1em;
	border: 3px solid #000000;
	border-radius: 10px 0 15px 30px;
	box-shadow: 10px 10px 5px 5px #888888;
}
article {
	border-left: 3px double #000000;
	padding-left: 1em;
	border-radius: 30px;
}

Output:

This section has a black solid border with rounded corners.

This article has a left double border with large rounded corneres.
