The CSS Box Model

An example of the CSS Box Model

HTML

<section>
	<h1>CSS Box Model</h1>
	<p>This is an example of the CSS Box Model.
	   All block elements get put in a box.</p>
</section>

CSS

section {
	border: 2px solid #000000;
	width: 500px;
	margin: 20px;	/* All Four Sides */
	padding: 10px;	/* All Four Sides */
}
h1,
p {
	border: 1px dashed #000000;
	padding-left: 15px;
}
h1 {
	margin: .5em 0 .25em;	/* .5em top, 0 right and left, .25em bottom */
}
p {
	margin: 0;	/* All Four Sides */
}

Output:

CSS Box Model

This is an example of the CSS Box Model. All block elements get put in a box.
