CSS Borders

Basic Border

HTML

<section>
	<h1>CSS Box Model</h1>
	<p>This is an example of the CSS Box Model.
	   All block elements get put in a box.</p>
</section>

CSS

section {
	border: 1px dashed #000000;
}

Output:

CSS Box Model

This is an example of the CSS Box Model. All block elements get put in a box.

Other Borders

HTML

<section>
	<h1>CSS Box Model</h1>
	<p>This is an example of the CSS Box Model.
		 All block elements get put in a box.</p>
</section>

CSS

section p {
	border-left: 3px double #000000;
	padding-left: 1em;
}

section p:hover {
	border-left-color: #FF0000;
	border-left-width: 16px;
}

Output:

CSS Box Model

This is an example of the CSS Box Model. All block elements get put in a box.
