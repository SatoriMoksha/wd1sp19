The CSS Box Model

A Simple Box

HTML

<section class="simple">
  <p>I'm a simple box</p>
</section>

CSS

.simple {
	width: 500px;
	margin: 20px auto;
}

Output:

I'm a simple box

A Fancy Box

HTML

<section class="fancy">
	<p>I'm a fancy box</p>
</section>

CSS

.fancy {
	width: 500px;
	margin: 20px auto;
	padding: 50px;
	border-width: 10px;
	border-color: #0000FF;
	border-style: dashed;
}

Output:

I'm a fancy box

vs.

I'm a simple box

A Simple Box, Ignoring the Box Model

HTML

<section class="simple-border-box">
  <p>I'm a simple box that doesn't follow the box model!</p>
</section>

CSS

.simple-border-box {
	width: 500px;
	margin: 20px auto;
	box-sizing: border-box;
}

Output:

I'm a simple box that doesn't follow the box model!

A Fancy Box, Ignoring the Box Model

HTML

<section class="fancy-border-box">
	<p>I'm a fancy box that doesn't follow the box model!</p>
</section>

CSS

.fancy-border-box {
	width: 500px;
	margin: 20px auto;
	padding: 50px;
	border-width: 10px;
	border-color: #0000FF;
	border-style: dashed;
	box-sizing: border-box;
}

Output:

I'm a fancy box that doesn't follow the box model!

vs.

I'm a fancy box

vs.

I'm a simple box that doesn't follow the box model!

vs.

I'm a simple box

Ignore the Box Model for Entire Website

HTML

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<meta http-equiv="X-UA-Compatible" content="ie=edge">
	<title>Document</title>
</head>
<body>

</body>
</html>

CSS

html {
  box-sizing: border-box;
}
*, *:before, *:after {
  box-sizing: inherit;
}
