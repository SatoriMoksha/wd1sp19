CSS Font Families

Basic Font Families

HTML

<p class="serif">This is a Serif Font</p>
<p class="sans-serif">This is a Sans-Serif Font</p>
<p class="mono">This is a Monospace Font</p>
<p class="fantasy">This is the Default Fantasy Font</p>
<p class="cursive">This is the Defult Cursive Font</p

CSS

.serif {
  font-family: "Times New Roman", times, serif;
}

.sans-serif {
  font-family: Helvetica, Arial, sans-serif;
}

.mono {
  font-family: "Courier New", Courier, monospace;
}

.fantasy {
  font-family: fantasy;
}

.cursive {
  font-family: cursive;
}

Output:

This is a Serif Font

This is a Sans-Serif Font

This is a Monospace Font

This is the Default Fantasy Font

This is the Defult Cursive Font

Embed a New Font Family

HTML

<head>
  <link href="https://fonts.googleapis.com/css?family=Press+Start+2P" rel="stylesheet">
</head>
<body>
  <p>This text will use the embeded font<p>
</body>

CSS

.serif {
  font-family: "Times New Roman", times, serif;
  }

  .sans-serif {
  font-family: Helvetica, Arial, sans-serif;
  }

  .mono {
  font-family: "Courier New", Courier, monospace;
  }

Output:

This text will use the embeded font
