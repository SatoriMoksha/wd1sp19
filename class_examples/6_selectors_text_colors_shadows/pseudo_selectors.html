CSS Pseudo Selectors

Pseudo-Class for Links Selectors

HTML

<ul>
  <li><a class="one" href="#">Link 1</a></li>
  <li><a class="two" href="#">Link 2</a></li>
  <li><a class="three" href="#">Link 3</a></li>
  <li><a class="four" href="http://www.google.com">Google</a></li>
</ul>

CSS

/* Non-Visited Link */
a:link {
  color: #FF0000;
}
/* Visited Link */
a:visited {
  color: #228B22;
}
/* Click but not released */
a:active {
  font-weight: bold;
}
/* Hovering or Focus */
a:hover,
a:focus {
  color: #1E90FF;
}

Output:

  • Link 1
  • Link 2
  • Link 3
  • Google

Pseudo-Class :not Selector

HTML

<div>A Div</div>
<div class="awesome">An Awesome Div</div>
<section>A Section</section>
<section class="awesome">An Awesome Section</section>

CSS

:not(.awesome) {
  font-variant: small-caps;
  font-weight: bold;
}
div:not(.awesome) {
  background: yellow;
}

Output:

A Div
An Awesome Div
A Section
An Awesome Section

Pseudo-Class child Selectors

HTML

<ul>
  <li>First Child</li>
  <li>Middle Child</li>
  <li>Middle Child</li>
  <li>Last Child</li>
</ul>
<ul>
  <li>First, Middle and Only Child</li>
</ul>

CSS

/* The first child of an element */
ul li:first-child{
  font-weight:bold;
}
/* The last child of an element */
ul li:last-child{
  word-spacing: 3em;
}
/* The only child of an element */
ul li:only-child{
  background: yellow;
}

Output:

  • First Child
  • Middle Child
  • Middle Child
  • Last Child
  • First, Middle and Only Child

CSS

p::first-letter {
  font-size: 200%;
  font-weight: bold;
}

p::first-line {
  background: yellow;
}

p::selection {
  /* Can only use color, background,
     background-color, and text-shadow */
  color: red;
}

Pseudo-Element Generated Content Selectors

HTML

<a href="http://google.com/">Search the Web</a><br>
<a class="before" href="http://google.com/">Search the Web</a><br>
<a class="after" href="http://google.com/">Search the Web</a>

CSS

a.after::after {
  color: #9799a7;
  content: " (" attr(href) ")";
  font-size: 11px;
}

a.before::before {
  content: "♥ ";
}

Output:

Search the Web
Search the Web
Search the Web
