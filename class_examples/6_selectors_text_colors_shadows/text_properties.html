CSS Text Properties and Values

Some Text Properties

HTML

<p class="font-size">Changing the Font Size</p>
<p class="font-style">Changing the Font Style</p>
<p class="font-weight">Changing the Font Weight</p>
<p class="font-variant">Changing the Font Variant</p>
<p class="line-height">Changing the<br>Line Height</p>

CSS

.font-size {
  font-size: 3em;
}

.font-style {
  font-style: italic;
}

.font-weight {
  font-weight: bold;
}

.font-variant {
  font-variant: small-caps;
}

.line-height {
  line-height: 3em;
}

Output:

Changing the Font Size

Changing the Font Style

Changing the Font Weight

Changing the Font Variant

Changing the
Line Height

Text Alignment Properties

HTML

<p class="indent">This first line is indented!<br>I am not!</p>
<p class="text-align-middle">I am aligned in the center</p>
<p class="text-align-right">I am aligned to the right</p>

CSS

.indent {
  text-indent: 2em;
}

.text-align-middle {
  text-align: center;
}

.text-align-right {
  text-align: right;
}

Output:

This first line is indented!
I am not!

I am aligned in the center

I am aligned to the right

More Text Properties

HTML

<p class="text-transform">transforming the text</p>
<p class="text-decoration">Adding text decoration</p>
<p class="letter-spacing">These Letter Have Spacing!</p>
<p class="word-spacing">These Words Have Spacing!</p>

CSS

.text-transform {
  text-transform: uppercase;
}

.text-decoration {
  text-decoration: underline;
}

.letter-spacing {
  letter-spacing: 1.5em;
}

.word-spacing {
  word-spacing: 3em;
}

Output:

transforming the text

Adding text decoration

These Letter Have Spacing!

These Words Have Spacing!

Text Shadows

HTML

<p class="text-shadow-1">A Text Shadow</p>
<p class="text-shadow-2">Another Text Shadow</p>
<p class="text-shadow-3">A Third Text Shadow</p>

CSS

.text-shadow-1 {
  text-shadow: 1px 1px 2px #000000;
}

.text-shadow-2 {
  text-shadow: -2px -2px 4px #FF0000;
}

.text-shadow-3 {
  text-shadow: 4px 4px;
}

Output:

A Text Shadow

Another Text Shadow

A Third Text Shadow
