The CSS Box Model

Relative Positioning

HTML

<section></section>
<section><span>Relative</span></section>
<section></section>

CSS

section {
	width: 100px;
	height: 100px;
	background: deepskyblue;
	margin: .25em 0;
}

section span {
	position: relative;
	display: block;
	width: 100px;
	height: 100px;
	top: 20px;
	left: 20px;
	background: forestgreen;
}

Output:

Relative

Absolute Positioning

HTML

<section>
	<h4>Absolute</h4>
	<h5>Over Absolute</h5>
	<p>Just Some Regular Text</p>
</section>

CSS

h4 {
	position: absolute;
	top: 50px;
	right: 100px;
	background: black;
	color: #FFFFFF;
	z-index: 1;
	height: 100px;
}

h5 {
	position: absolute;
	right: 50px;
	top: 50px;
	z-index: 2;
	background: #FFFFFF;
	border: 2px solid #000000;
	padding: .5em;
}

Output:

Absolute

Over Absolute

Just Some Regular Text

Fixed Positioning

HTML

<footer>
	<p>This footer is fixed to the bottom of the page</p>
</footer>

CSS

footer {
	position: fixed;
	bottom: 0;
	height: 50px;
	background: #000000;
	color: #FFFFFF;
	text-align: center;
	width: 100%;
}

Output:

This footer is fixed to the bottom of the page
