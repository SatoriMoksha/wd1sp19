CSS Floats

An example of the CSS Floats

HTML

<section>
	<p class="left">Floated Left</p>
	<p class="right">Floated Right</p>
	<p>No Float, <br>
	   Should be wrapping around<br>
	   the left float</p>
</section>

CSS

.left {
	float: left;
	margin-right: 10px;
	background: #EBEBEB;
}
.right {
	float: right;
	background: #BEBEBE;
}
section p:last-child {
	background: tomato;
}

Output:

Floated Left

Floated Right

No Float,
Should be wrapping around
the left float

An example of (Not) Clearing Floats

HTML

<header>
  <code>&#60;header&#62;</code>
</header>
<section>
  <code>&#60;section&#62; <br> float: left;</code>
</section>
<aside>
  <code>&#60;aside&#62; <br> float: right;</code>
</aside>
<footer>
  <code>&#60;footer&#62;</code>
</footer>

CSS

section {
  float: left;
}
aside {
  float: right;
}

Output:

<header>
<section>
float: left;
<aside>
float: right;
<footer>

An example of Clearing Floats

HTML

<header>
<code>&#60;header&#62;</code>
</header>
<section>
<code>&#60;section&#62; <br> float: left;</code>
</section>
<aside>
<code>&#60;aside&#62; <br> float: right;</code>
</aside>
<footer>
<code>&#60;footer&#62;</code>
</footer>

CSS

section {
  float: left;
}
aside {
  float: right;
}
footer {
	clear: both:
}

Output:

<header>
<section>
float: left;
<aside>
float: right;
<footer>
